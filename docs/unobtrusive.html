CSS Grid and Unbtrusive Styling

The new css Grid makes possible the ability to generate web page layouts in an unobtrusive way.

What does this mean for HTML markup? First of all, it means that our HTML markup can be a lot cleaner. No more nesting content inside of <div> tags, or littering our markup with classes. Secondly, it means that our HTML can be semantic: the structure of our markup should focus on the meaning of the content.
